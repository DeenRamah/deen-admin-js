## 🚀 Getting Started

### 🧬 Clone the Repository

```bash
git clone https://github.com/DeenRamah/deen-admin-js.git
cd deen-admin-js
```

### 📦 Install Dependencies

Using **npm**:

```bash
npm install
```

Or with **yarn**:

```bash
yarn install
```

Or with **pnpm**:

```bash
pnpm install
```

### 🔧 Run in Development

```bash
npm run dev
```

Open your browser at [http://localhost:3000](http://localhost:3000) to see the dashboard.
